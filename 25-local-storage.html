LocalStorage

Local Storage é uma propriedade do objeto window que pode armazenar dados no computador do usuário e servem para lembrar suas informações, sempre que ele visitar o site. Podemos guardar o seu nome, e-mail, suas preferências, histórico de utilização do site ou qualquer outra informação. (Passwords e informações sensíveis nunca devem ser guardados em local storage).

Informações devem ser armazenadas em local storage como duplas de chave e valor.

Armazenar dados

Usamos o método setItem para apagar uma informação armazenada em local storage. 

                    
    window.localStorage.setItem("nome", "ivan");
    // o objeto window pode ser omitido quando trabalhamos com o local storage
    
    // A consulta aos dados armazenados em local storage é semelhante à consulta de valores em um objeto.
    // Podemos usar a notação com ponto ou com colchetes
    console.log(localStorage['nome']);
Esta informação fica guardada até que seja apagada via código ou quando o usuário limpa o cache do navegador. Podemos encontrar todas as informações guardadas em local storage de um site pelo console, na aba Application, na seção Storage > Local Storage.

Remover dados

Usamos o método removeItem para apagar uma informação armazenada em local storage. 

                    
    localStorage.removeItem("nome");
